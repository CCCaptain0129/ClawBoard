## 实现内容

### 1. 任务执行耗时统计

- **开始时间**: 显示任务开始执行的相对时间（如 "1小时前"）
- **完成时间**: 完成任务时显示完成时间
- **总耗时**: 自动计算并显示执行时长（如 "1小时30分钟"）

### 2. 任务超时检测

- **自动识别**: 基于 `estimatedTime` 字段自动检测任务是否超时
- **视觉警告**: 超时任务使用红色警告样式（边框、图标、文字）
- **超时标签**: 显示 "⚠️ 执行超时" 标签提醒用户

### 3. 执行日志展示

- **日志摘要**: 显示最新一条执行记录的摘要（前50字符）
- **日志计数**: 显示日志总数（如 "执行日志 (5条)"）
- **独立卡片**: 使用黄色卡片样式区分于 Subagent 信息

### 4. Subagent 执行状态可视化

- **进行中**: 蓝色脉冲效果 + "🔄 执行中" 标签
- **超时**: 红色警告样式 + "⚠️ 执行超时" 标签
- **完成**: 绿色耗时显示 + 完成时间提示
- **Subagent ID**: 短化显示（只显示最后12位）

### 5. 预计执行时间提示

- **实时显示**: 进行中的任务显示预计完成时间
- **格式化**: 支持多种格式（"30分钟"、"1小时"、"2小时"）

## 技术实现

### 接口扩展

```typescript
interface Task {
  // 原有字段...
  dueDate?: string | null;          // 截止日期
  estimatedTime?: string | null;    // 预计执行时间

```

### 新增工具函数

1. **calculateDuration(startTime, completeTime)**: 计算执行耗时
2. **isTaskOverdue(task)**: 检测任务是否超时
3. **formatLogSummary(comments)**: 格式化日志摘要
